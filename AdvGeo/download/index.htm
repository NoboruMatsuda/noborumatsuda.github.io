Downloading and installing Advanced Geometry Tutor

If you are interested, Advanced Geometry Tutor (AGT) is available for an academic use.

Unfortunately, though, AGT is not initially developed for a wide distribution, and you would need a heck of knowledge in running LISP and Java programs as well as tuning the run-time environment variables involved in the software package.

AGT is written in LISP and Java.

AGT.class is the Java class to run the tutor interface (GUI).

LISP modules are for the core modules of the tutor.   To run the LISP modules, we need to have Allegro Common Lisp 6.2 or other compatible versions.

You need to have a Lisp module and a Java module running at the same time to run AGT. The Lisp and Java modules communicate each other through a socket stream channel.

Downloading files

The entire AGT modules are distributed as two ZIP'd files

download lisp.zip (880KB) and AGT.zip (770KB)

Installing files

Now, make a folder, say AdvGeoTutor, and unzip two files right underneath. You should have two folders lisp and AGT as noted below.

AdvGeoTutor
      AGT
            Java
            lisp
            Perl
      lisp
            bat
            DS
            Log
            memo
            ...

Delete all .fasl files in the "lisp" folder.

Running AGT

You may need to modify lisp/gramy.lisp, AGT/lisp/site.lisp, and AGT/lisp/AGT.lisp to specify appropriate parameters. Look for "defparameter" statements and change the value.  For example, if you have the AGT folder at "C:/MyThesis/AGT" then change (defparameter *AGT-DIR* "f:/Home/Gramy/AGT/lisp/") to (defparameter *AGT-DIR* "C:/MyThesis/AGT/lisp/")

(*) If you have trouble compiling LISP files, then don't even bother compiling them. All what you need is to load boot.lisp.

You may also need to place an absolute path in boot.lisp

Open AllegroCL and change the default package from COMMON-GRAPHICS-USER to COMMON-LISP-USER

To run the Lisp part, you need to just load AGT/lisp/boot.lisp, and run (AGT:start-AGT-server) from the LISP interpreter. Notice the parentheses (why, it's LISP!).

Modify commanager.java to use a localhost instead of vanlehn22.lrdc.pitt.edu.  Recompile commanager.java

After launching the AGT-server (from the List interpreter), you need to launch the AGT interface, by just double-clicking AGT/Java/AGT.html
