  • GRAMY-new-search

A basic idea for the next experiment

  • The search with brute-force construction might work for a simple problem.  In that case, the searches with brute-force construction and the general construction must produce exactly the same set of proofs.
  • The search with general construction might work for more difficult problems as well, but it might fail on "regular" problems such that the ones we saw in the literature.
  • For the problems that beyond the general construction, we need to modify the construction technique so that the conditions for construction are embedded into DS matching (like current GRAMY's construction operator).  

Lessons learned

  • Once we built an AI system, we need to think if what we got is really what we need.  To see that we must appreciate the "knowledge" implemented in the system.  In the present study, it is the heuristics for construction that we should realize the value.  If you can't see their value (which is usually the case), losing them makes you become conscious of their true value.
