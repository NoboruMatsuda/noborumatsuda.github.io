Interesting Questions to Explore

How to get the bottom-out hint work?

Most failures on the bottom-out hint might be bugs to be fixed.  Pilot test must be carried out. 

I have a second thought on this question:  how do the human tutors avoid using bottom-out hint?   In other words, how do human tutors make students understand things that ITS can't teach?  Indeed, human teachers hardly invoke a bottom-out hint: it would be very silly if a human tutor says like "put this equation there, anyway!" 

Which is better to provide FC hint or BC hint?

Scheines CPT shows that it is beneficial for students both FC and BC be implemented in the tutor.  

Hence, we would have to investigate the question when we should provide FC (BC) hint and why?

What kind of scaffolding do students really need? 

Seems like using a proof tree (or modeling a proof as search) doesn't help students to learn proof writing.  But, what else can we provide?

If a student really understands the structure of proofs, then he/she should write proofs.  This is because writing geometry proofs per se (i.e., building up a chain of postulate applications) shouldn't be too complicated.  Hence, we must invent a better cognitive model that represents what students really need.  

more questions:

Does abstraction before execution really help students to learn proof writing?
